# Hero Section Images

Place your vegetable/fruit images in this folder to replace the placeholder images.

## Required Images

### Main Showcase Images (Larger, rotating in center)
These should be high-quality, square images (recommended: 400x400px or larger)

- `tomato.png` - Fresh tomatoes
- `lettuce.png` - Fresh lettuce/greens
- `carrot.png` - Fresh carrots

### Floating Accent Images (Smaller, decorative)
These should be smaller, square images (recommended: 200x200px)

- `apple.png` - Apple
- `pepper.png` - Bell pepper
- `broccoli.png` - Broccoli
- `orange.png` - Orange
- `avocado.png` - Avocado
- `lemon.png` - Lemon

## Image Guidelines

1. **Format**: PNG with transparent background works best
2. **Size**: 
   - Main images: 400x400px minimum
   - Floating images: 200x200px minimum
3. **Quality**: High resolution for crisp display on retina screens
4. **Style**: Clean, well-lit product photography

## Fallback Images

If images are not found, the component will automatically use placeholder images from Unsplash. Once you add your own images, they will be used instead.
